# TP4: Appearance-Aware IoU-Kalman Object Tracker

## Overview

Extends TP3 by adding an Object Re-Identification (ReID) module to improve
association using visual features.

## Requirements

- `scipy`
- `opencv-python` (with DNN support)
- `ADL-Rundle-6` dataset (MOT15)
- `reid_osnet_x025_market1501.onnx` (in project root)

## Setup

(Same as TP2/3)

## Running

```bash
python tp4/main.py
```
